## 🌐 Overview
**Terranis** is a high-end, single-file geospatial observatory dashboard that aggregates live telemetry streams for natural hazards and planetary health metrics. It tracks seismic movements, severe river discharge, extreme heat spikes, and global warming indicators across all continents in real time.

---

## ⚡ Core Features
* **Interactive Global Matrix:** Dynamic Leaflet-powered mapping with custom risk-tiered hazard markers.
* **Multi-Agency Telemetry Feeds:** Direct integration with USGS (earthquakes), NOAA (meteorology), and Copernicus (floods).
* **Live Climate Tracking:** Continuous monitoring of global temperature anomalies, atmospheric $\text{CO}_2$, ocean heat content, and Arctic sea ice.
* **Instant Crisis Wire:** Live RSS-driven field coverage paired with complete searchable archives.
